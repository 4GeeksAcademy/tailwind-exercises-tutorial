# `07` Tailwind Buttons, Alert, and Table

Los botones, bloques tipo alerta, utilidades de color y estilos de tabla pueden tener diferentes propósitos visuales según el contexto.

![Example Image](../../.learn/assets/07-buttons-alert-and-table-img1.png?raw=true)

No es solo estética, cada estilo debe servir un propósito específico en tu sitio web.

## 📝 Instrucciones:

Replica exactamente esta misma imagen utilizando elementos de Tailwind CSS:

- Botones.
- Un bloque tipo alerta.
- Una tabla.
- También, sin falta, un layout de 2 columnas.

![Example Image](../../.learn/assets/07-buttons-alert-and-table-result.png?raw=true)

## 💡 Pista:

+ ¡No necesitas CSS personalizado!
