# `05` Tailwind Sidebar With Menu

## 📝 Instructions:

1. Replicate this exact design:

![Example Image](../../.learn/assets/05-sidebar-with-menu-result.png?raw=true)

2. The first column should use `w-1/6` and the second one should use `w-5/6`.

3. The title of the second column should be an `<h4>`.

## 💡 Hints:

+ Create a flex row with two columns. The left column must have a sidebar menu inside.

+ Don’t forget the background color in the second column (it’s a very light gray).
