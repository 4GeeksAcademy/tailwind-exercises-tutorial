# `05` Tailwind Sidebar With Menu

## 📝 Instrucciones:

1. Replica este diseño:

![Example Image](../../.learn/assets/05-sidebar-with-menu-result.png?raw=true)

2. La primera columna debe usar `w-1/6` y la segunda columna debe usar `w-5/6`.

3. El título de la segunda columna debe ser un `<h4>`.

## 💡 Pistas:

+ Crea una fila con `flex` y dos columnas. La columna izquierda debe tener una barra lateral con menú dentro.

+ No olvides poner el color de fondo de la segunda columna (es un gris muy claro).
 
## 🔎 Importante:

Estos documentos pueden ser útiles:

- https://tailwindcss.com/docs/flex
- https://tailwindcss.com/docs/width
- https://tailwindcss.com/docs/background-color
- https://tailwindcss.com/docs/padding
