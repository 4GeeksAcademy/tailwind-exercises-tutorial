# `08` Tailwind Forms

## 📝 Instrucciones:

1. Usa estilos de formulario de Tailwind para recrear exactamente el mismo formulario de inicio de sesión:

![Example Image](../../.learn/assets/08-bootstrap-forms-result.png?raw=true)

2. El formulario debe tener un color de fondo gris oscuro.

3. El formulario debe tener bordes redondeados.

4. El título debe ser un `<h2>`.

5. El input del email debe tener `type="email"`.

6. El input del password debe tener `type="password"`.

7. El input del checkbox debe tener `type="checkbox"`.

8. El botón debe tener un ancho de 100%.

9. Todo debe ser hecho con clases de utilidad de Tailwind (No se permite usar CSS personalizado).

## 💡 Pista:

Para hacer este ejercicio puedes usar la documentación de Tailwind:

- Formularios básicos: https://tailwindcss.com/docs/placeholder-color
- Bordes redondeados: https://tailwindcss.com/docs/border-radius
- Ancho: https://tailwindcss.com/docs/width
